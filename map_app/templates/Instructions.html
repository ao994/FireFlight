Instructions

Below, is an interactive tool that displays the health of an ecosystem. The heat map shows the amount of birds in a given area. There are filters on the side of map that will allow for there to be different aspects to be viewed separately or together.

  • Click the select all to view all birds.
  • Click deselct all to uncheck the birds in the filter.
  • Enter the desired bird species into the search bar.
  • Click the + and - to zoom in and out.
  • Click view larger map to get the map in full screen mode.
  • Click update to update the map based on filtered bird values.
  • Click export data to export bird data into a comma seperated values file (CSV)
